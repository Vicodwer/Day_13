# Pandas Data Analysis Assignment

## Overview

This assignment demonstrates the use of **Python Pandas for data manipulation and analysis**.
The objective is to perform common data analysis operations such as filtering, aggregation, sorting, and grouping using the Pandas library.

The implementation focuses on **clear, readable code and step-by-step data transformations** to replicate typical data analyst workflows.

---

## Learning Objectives

* Understand the Pandas DataFrame structure
* Perform filtering and selection of data
* Apply aggregation and grouping operations
* Sort and rank datasets
* Interpret data insights from structured datasets

---
