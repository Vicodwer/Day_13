# Data Cleaning and Preprocessing Assignment

## Overview

This assignment focuses on **data cleaning and preprocessing techniques** using Python and Pandas.
Real-world datasets often contain missing values, duplicates, inconsistent formats, and outliers.
The objective of this assignment is to transform raw data into a **clean, analysis-ready dataset**.

---

## Learning Objectives

* Handle missing values
* Detect and remove duplicate records
* Standardize inconsistent data formats
* Identify and manage outliers
* Prepare datasets for analysis and machine learning

---

## Technologies Used

* Python 3.x
* Pandas
* NumPy
* Jupyter Notebook
* Git & GitHub
